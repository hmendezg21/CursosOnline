# ⚡ Electrónica 226 UPG

Bienvenido al curso **Electrónica 226 UPG**.

En este curso aprenderás los fundamentos de la electrónica analógica mediante ejemplos prácticos y explicaciones paso a paso, abordando el funcionamiento de diodos, transistores y circuitos electrónicos básicos.

# 🎯 Objetivos del curso

Al finalizar este curso serás capaz de:

* ⚡ Comprender el funcionamiento del diodo.
* 🔌 Analizar circuitos con diodos.
* 📈 Entender el comportamiento del diodo Zener.
* 🔄 Utilizar transistores como interruptores.
* 📊 Analizar el punto de operación de un transistor.
